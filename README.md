# Practica-1-de-PHP
En este repositorio se incluyen 3 problemas para probar la programación en PHP

El problema 1 consiste en la impresión de la línea "Hello World!"

El problema 2 incluye 6 operaciones matemáticas: suma, resta, multiplicación, división, módulo, y potencia.

Por último, el problema 3 incluye 2 problemas con orden de procedencia. El primero consiste en una suma con multiplicación, y el segundo una suma entre parentesis con una multiplicación.
